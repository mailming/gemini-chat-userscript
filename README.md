# Gemini Chat Userscript

A userscript for enhancing the Gemini chat experience at gemini.google.com

## Installation

1. Install a userscript manager like [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/)
2. Open the userscript manager and create a new script
3. Copy the contents of `gemini-chat.user.js` into the editor
4. Save the script

## Usage

The script will automatically run when you visit gemini.google.com
